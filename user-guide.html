ĐĂNG NHẬP

Hỗ trợ nhiều phương thức đăng nhập nhanh chóng và bảo mật.

📧 Email & Password

Đăng ký tài khoản mới qua mục "Register" hoặc đăng nhập tài khoản đã có.

🔵 Google

Đăng nhập nhanh bằng tài khoản Google có sẵn trên điện thoại.

🍎 Apple ID

Đăng nhập bằng Apple ID (iOS). Bảo mật cao với Face ID / Touch ID.

📱 Số điện thoại

Đăng nhập bằng SĐT + OTP. Xác thực 2 lớp an toàn.

3

THÊM THIẾT BỊ GATEWAY

Mở App → nhấn nút "+"
Tại màn hình chính, nhấn nút thêm thiết bị ở góc phải.
Quét mã QR trên Gateway
Mỗi Gateway Navis MOD có mã QR dán trên mặt thiết bị. Dùng camera trong app để quét.
Nhập Code (nếu không quét QR)
Nhập mã code được cung cấp trong tài liệu bàn giao hệ thống.
Đặt tên trạm
Đặt tên cho trạm (VD: "Trạm xử lý nước KCN A") để dễ nhận biết.
Hoàn tất
Gateway sẽ hiện trên Dashboard. Dữ liệu cập nhật realtime sau vài giây.

CẤU HÌNH MODBUS QUA APP

Vào Settings → Advance Settings
Chọn thiết bị → Settings → Advance Settings → Modbus Settings.
Chọn template hoặc nhập thủ công
Chọn template có sẵn cho thiết bị phổ biến, hoặc nhập: Slave ID, Function Code, Address, Data Type.
Lưu & Cập nhật
Nhấn Lưu → Update. Đợi 20–30 giây để gateway nhận cấu hình mới.

💡 Xem hướng dẫn chi tiết: Hướng dẫn kết nối Modbus từ A-Z →

7

XỬ LÝ SỰ CỐ THƯỜNG GẶP

Không nhận Push Notification

Kiểm tra quyền thông báo trong Cài đặt điện thoại → Navis → Notifications → Bật "Allow Notifications".

App không hiện dữ liệu

Kiểm tra Gateway có online không (LED sáng liên tục). Kéo xuống để refresh. Kiểm tra kết nối mạng điện thoại.

Quét QR không nhận

Đảm bảo camera có đủ ánh sáng. Thử nhập Code thủ công thay vì quét QR.

Không đăng nhập được

Kiểm tra kết nối mạng. Thử đăng nhập bằng phương thức khác (Google/Apple). Liên hệ hỗ trợ nếu tài khoản bị khóa.
