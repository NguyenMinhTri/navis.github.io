3.2 Thao Tác Nhanh

Thao tác Chức năng
Nhấn vào item Mở form chỉnh sửa
Vuốt phải → Nhân bản cấu hình
Vuốt trái ← Xóa cấu hình
Kéo thả ↕ Sắp xếp thứ tự
Nút + (góc phải) Thêm mới
Nút Save 💾 Lưu thay đổi xuống Gateway

3.3 Giải Thích Các Trường Cấu Hình

Trường Mô tả Ví dụ
Name Tên hiển thị (≤30 ký tự, không chứa /) Nhiệt độ lò
Address Địa chỉ Modbus (thập phân) 4196 (D100)
Slave ID ID của thiết bị slave (1-247) 1
Group Name Tên nhóm (header, ≤30 ký tự) PLC_Delta
Function Code Mã chức năng Modbus 1, 2, 3, 4
Type Kiểu dữ liệu bool, int, float
Memory Type Loại hiển thị trên app 0: COIL, 1: SENSOR
Unit Đơn vị hiển thị °C, bar, %, Hz
Changing Unit Hệ số nhân giá trị 0.1 (255→25.5)
Fractional Digit Số chữ số thập phân 0, 1, 2
Delta Value Ngưỡng thay đổi gửi data 0.1
Cycle Time Chu kỳ đọc (ms, ≥1) 1000
Offset Cộng thêm vào giá trị 0
Get Bit Lấy bit cụ thể (-1 = không) -1, 0, 1...
Is Modify Cho phép ghi xuống PLC ✓
Is High Alarm Hiển thị màu đỏ khi ON ✓
Is Hide Ẩn khỏi giao diện ✓
Is Column Hiển thị dạng cột ✓
Analog Scale Quy đổi In→Out 0-4095→0-100
