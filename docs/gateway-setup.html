HƯỚNG DẪN CÀI ĐẶT

Gateway Navis MOD

Hướng dẫn từng bước kết nối và cấu hình Gateway thu thập dữ liệu Modbus RTU/TCP

1

TỔNG QUAN

Nguồn12–24 VDC
Giao tiếpRS485 · TCP/IP
Kết nốiWiFi · LAN · 4G
Kích thước10 × 7.5 × 3 cm
Tối đa100 tag
Giao thứcModbus RTU/TCP
2

SƠ ĐỒ NỐI DÂY

Kết nối đúng cực A↔A, B↔B. Sai cực sẽ không gây hỏng nhưng không đọc được dữ liệu.

3

KẾT NỐI MẠNG

📶 WiFi

Giữ nút Config >5 giây → Mở App → Kết nối WiFi 2.4 GHz

🔌 LAN (Ethernet)

Cắm cáp mạng RJ45 → Nhấn 1 lần nút Config → Tự nhận IP (DHCP)

📡 4G (chỉ Việt Nam)

Lắp SIM 4G vào khe → Cấp nguồn → Gateway tự kết nối

4

CẤU HÌNH MODBUS QUA APP

Mở App Navis Smart Industrial
Tải trên Google Play hoặc App Store, đăng nhập tài khoản.
Quét QR Code
Quét mã QR trên mặt gateway để thêm thiết bị vào hệ thống.
Advance Settings → Modbus Settings
Vào Settings → Advance Settings → Modbus Settings để cấu hình giao thức.
Cấu hình thanh ghi
Chọn template có sẵn hoặc nhập thủ công: Slave ID, Function Code, Address, Data Type.
Lưu & Cập nhật
Nhấn Lưu → Update. Đợi 20–30 giây để gateway nhận cấu hình mới.

6

XỬ LÝ SỰ CỐ

Gateway không online

Kiểm tra nguồn 12–24VDC, kết nối mạng (WiFi/LAN/4G), và tín hiệu LED. Thử restart gateway.

Không đọc được Modbus

Kiểm tra đấu dây A/B đúng cực, baudrate khớp (9600/19200/115200), Slave ID chính xác, và khoảng cách cáp RS485 (<1200m).

LED nhấp nháy liên tục

Gateway đang trong quá trình kết nối. Đợi tối đa 60 giây. Nếu vẫn nhấp nháy, kiểm tra lại cấu hình mạng.
